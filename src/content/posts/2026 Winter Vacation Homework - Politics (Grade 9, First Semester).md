1. 当代中国最鲜明的特色是 <span style="color:red; background:yellow">改革开放</span>；党和政府坚持 <span style="color:red; background:yellow">以人民为中心</span> 的发展思想；党的奋斗目标是 <span style="color:red; background:yellow">人民对美好生活的向往</span>；发展的根本目的是 <span style="color:red; background:yellow">增进民生福祉</span>。
2. <span style="color:red; background:yellow">1978</span> 年，党的 <span style="color:red; background:yellow">十一届三中全会</span> 开启了改革开放的历史征程。
3. <span style="color:red; background:yellow">经济建设</span> 是我国当前的中心工作、一切工作的中心。
4. 中国特色社会主义的本质要求是 <span style="color:red; background:yellow">共同富裕</span>；中国特色社会主义的本质要求和重要保障是 <span style="color:red; background:yellow">全面依法治国</span>。<span style="color:red; background:yellow">习近平法治思想</span> 是全面依法治国的根本遵循和行动指南。
5. 综合国力竞争的决定性因素是 <span style="color:red; background:yellow">科技创新能力</span>；改革开放的生命是 <span style="color:red; background:yellow">创新</span>。创新是发展的第一 <span style="color:red; background:yellow">动力</span>。科技是第一 <span style="color:red; background:yellow">生产力</span>；人才是第一 <span style="color:red; background:yellow">资源</span>。
6. 民族振兴、社会进步的基石，促进人的全面发展的根本途径是 <span style="color:red; background:yellow">教育</span>。
7. 人民掌握政权、行使权力的根本途径是 <span style="color:red; background:yellow">人民代表大会制度</span>；社会主义民主政治建设的基础是 <span style="color:red; background:yellow">基层群众自治制度</span>。
8. 社会主义民主政治的本质属性是 <span style="color:red; background:yellow">全过程人民民主</span>；本质特征是 <span style="color:red; background:yellow">人民当家作主</span>。
9. 全过程人民民主的根本保证是 <span style="color:red; background:yellow">党的领导</span>；特有形式和独特优势是 <span style="color:red; background:yellow">协商民主</span>。社会主义民主的真谛是 <span style="color:red; background:yellow">有事好商量，众人的事情由众人商量</span>。
10. 依法行政的核心是 <span style="color:red; background:yellow">规范政府的行政权</span>。
11. 现代法治政府行使权力普遍奉行的基本准则是 <span style="color:red; background:yellow">依法行政</span>。政府的宗旨是 <span style="color:red; background:yellow">为人民服务</span>。
12. 中华文化的精髓是 <span style="color:red; background:yellow">中华传统美德</span>；一个国家、一个民族的灵魂是 <span style="color:red; background:yellow">文化</span>。<span style="color:red; background:yellow">文化自信</span> 是一个国家、一个民族发展中最基本、最深沉、最持久的力量。
13. 当代中国精神的集中体现是 <span style="color:red; background:yellow">社会主义核心价值观</span>。
14. 当代中国人评判是非曲直的价值标准是 <span style="color:red; background:yellow">社会主义核心价值观</span>。
15. 自由、平等、公正、法治是 <span style="color:red; background:yellow">社会</span> 层面的价值取向；爱国、敬业、诚信、友善是 <span style="color:red; background:yellow">公民</span> 层面的价值准则；富强、民主、文明、和谐是 <span style="color:red; background:yellow">国家</span> 层面的价值目标。
16. 处理民族关系的方针是 <span style="color:red; background:yellow">民族平等、民族团结、各民族共同繁荣</span>。
17. 已经形成的社会主义民族关系是 <span style="color:red; background:yellow">平等、团结、互助、和谐</span>。
18. <span style="color:red; background:yellow">和平统一、一国两制</span> 是解决台湾问题的基本方针和最佳方式；<span style="color:red; background:yellow">一个中国原则</span> 是两岸关系的政治基础。
19. 三个基本国策分别是 <span style="color:red; background:yellow">对外开放、计划生育、节约资源和保护环境</span>。
20. 实现中国梦，就是要实现国家 <span style="color:red; background:yellow">富强</span>、<span style="color:red; background:yellow">民族振兴</span>、<span style="color:red; background:yellow">人民幸福</span>。
21. 第一个百年奋斗目标是 <span style="color:red; background:yellow">全面建成小康社会</span>；到2035 年，基本实现 <span style="color:red; background:yellow">社会主义现代化</span>；到2050 年，把我国建设成富强民主文明和谐美丽的 <span style="color:red; background:yellow">社会主义现代化强国</span>。
22. 中国道路就是中国 <span style="color:red; background:yellow">特色社会主义</span> 道路；中国精神就是以 <span style="color:red; background:yellow">爱国主义</span> 为核心的 <span style="color:red; background:yellow">民族精神</span> 和以 <span style="color:red; background:yellow">改革创新</span> 为核心的 <span style="color:red; background:yellow">时代精神</span>；中国力量就是 <span style="color:red; background:yellow">全国各族人民大团结</span> 的力量。
23. 中国特色社会主义道路、<span style="color:red; background:yellow">中国特色社会主义理论体系、中国特色社会主义制度、中国特色社会主义文化</span> 是中国自信、民族自信的根本所在。
24. 新发展理念: <span style="color:red; background:yellow">创新</span>、协调、<span style="color:red; background:yellow">绿色、开放、共享</span>。四个全面: <span style="color:red; background:yellow">全面建设社会主义现代化国家</span>、全面深化改革、全面依法治国、全面从严治党。五位一体: <span style="color:red; background:yellow">经济</span> 建设、<span style="color:red; background:yellow">政治</span> 建设、文化建设、社会建设、生态文明建设。
