1. <span style="color:red; background:yellow">中国共产党领导</span>是中国特色社会主义最本质的特征，是中国特色社会主义制度的最大优势。党的 <span style="color:red; background:yellow">宗旨</span> 是全心全意为人民服务。党的最高理想和最终目标是 <span style="color:red; background:yellow">实现共产主义</span>。
2. 我国宪法是 <span style="color:red; background:yellow">党的主张</span> 和<span style="color:red; background:yellow">人民意志</span> 的统一；我国宪法是 <span style="color:red; background:yellow">党和人民意志</span> 的集中体现，是国家的<span style="color:red; background:yellow">根本</span>法。
3. 我国的根本制度是 <span style="color:red; background:yellow">社会主义制度</span>；根本政治制度是 <span style="color:red; background:yellow">人民代表大会制度</span>。
4. 我国宪法的基本原则是 <span style="color:red; background:yellow">国家的一切权力属于人民、国家尊重和保障人权</span>。宪法规定的社会主义<span style="color:red; background:yellow">经济</span>制度奠定了国家权力属于人民的 <span style="color:red; background:yellow">经济</span> 基础。宪法规定的社会主义<span style="color:red; background:yellow">政治</span>制度明确了人民行使国家权力的 <span style="color:red; background:yellow">基本</span> 途径和形式。
5. 人民幸福生活是 <span style="color:red; background:yellow">最大的人权</span>。尊重和保障人权是立法活动的 <span style="color:red; background:yellow">基本要求</span>。
6. 我国的国家机构实行 <span style="color:red; background:yellow">民主集中制</span> 的原则；国家权力机关在国家机构中居于 <span style="color:red; background:yellow">主导</span> 地位。全国人民代表大会是 <span style="color:red; background:yellow">最高国家权力</span> 机关，在整个国家机关体系中居于 <span style="color:red; background:yellow">最高</span> 地位。
7. 我国宪法的核心价值追求是 <span style="color:red; background:yellow">规范国家权力运行以保障公民权利</span>。
8. 宪法是一切组织和个人的 <span style="color:red; background:yellow">根本活动</span> 准则。宪法是国家 <span style="color:red; background:yellow">法制统一</span> 的基础。宪法具有最高的 <span style="color:red; background:yellow">法律效力</span>。宪法是中国特色社会主义法律体系的 <span style="color:red; background:yellow">核心</span>。坚持依法治国首先要坚持 <span style="color:red; background:yellow">依宪治国</span>，坚持依法执政首先要坚持 <span style="color:red; background:yellow">依宪执政</span>。
9. 宪法的制定和修改更为严格，一方面 <span style="color:red; background:yellow">使得宪法的内容具有更广泛的民意基础</span>；另一方面 <span style="color:red; background:yellow">可以保障宪法的长期稳定性，使国家长治久安、社会健康发展</span>。
10. 在公权力的监督体系中，宪法监督制度具有 <span style="color:red; background:yellow">基础性</span> 意义。宪法规定，<span style="color:red; background:yellow">全国</span> 人大及其常委会行使监督宪法实施的职责，全国 <span style="color:red; background:yellow">人大常委会</span> 有权解释宪法和法律。
11. 选举权和被选举权是公民的一项 <span style="color:red; background:yellow">基本政治权利</span>，行使这项权利是公民参与管理国家和管理社会的 <span style="color:red; background:yellow">基础</span>。
12. 人身自由是公民最<span style="color:red; background:yellow">基本</span>、最<span style="color:red; background:yellow">重要</span>的权利。<span style="color:red; background:yellow">劳动</span>和<span style="color:red; background:yellow">受教育</span>既是基本权利也是基本义务。
13. 公民的基本权利:
(1)政治自由和权利:<span style="color:red; background:yellow">选举</span>和 <span style="color:red; background:yellow">被选举</span>权；<span style="color:red; background:yellow">政治自由</span>权。
(2)人身自由: <span style="color:red; background:yellow">人身自由</span> 不受侵犯；<span style="color:red; background:yellow">人格尊严</span> 不受侵犯；住宅不受侵犯；通讯自由和通讯秘密受法律保护。(人格尊严包括<span style="color:red; background:yellow">名誉权、荣誉权、肖像权、姓名权、隐私权</span>)
(3)社会经济和文化权利: <span style="color:red; background:yellow">财产</span>权；<span style="color:red; background:yellow">劳动</span>权；<span style="color:red; background:yellow">物质帮助</span>权；受 <span style="color:red; background:yellow">教育</span>权；文化权利。
(4)其他权利:平等权、宗教信仰自由权利；妇女、儿童和残疾人等特定人群的权利受宪法和法律的特殊保护等。
14. 公民的基本义务:
(1)遵守宪法和法律(保守 <span style="color:red; background:yellow">国家秘密</span>、爱护 <span style="color:red; background:yellow">公共财产</span>、遵守 <span style="color:red; background:yellow">劳动纪律和公共秩序</span>、尊重 <span style="color:red; background:yellow">社会公德</span> 等)
(2)维护国家利益:(维护 <span style="color:red; background:yellow">国家统一和全国各民族团结</span>；维护 <span style="color:red; background:yellow">国家的安全、荣誉和利益</span>)
(3) <span style="color:red; background:yellow">依法服兵役</span>
(4) <span style="color:red; background:yellow">依法纳税</span>
(5)其他义务:<span style="color:red; background:yellow">劳动</span>的义务，<span style="color:red; background:yellow">受教育</span>的义务。夫妻双方实行计划生育的义务；父母抚养教育未成年子女的义务、成年子女赡养扶助父母的义务。
15. 人民代表大会制度是坚持 <span style="color:red; background:yellow">党的领导、人民当家作主、依法治国</span> 有机统一的根本政治制度安排。民族自治地方的 <span style="color:red; background:yellow">人民代表大会</span> 和 <span style="color:red; background:yellow">人民政府</span> 是自治机关。
16. <span style="color:red; background:yellow">中国人民政治协商会议</span> 是爱国统一战线组织。围绕<span style="color:red; background:yellow">团结</span>和 <span style="color:red; background:yellow">民主</span> 两大主题，履行 <span style="color:red; background:yellow">政治协商、民主监督、参政议政</span> 的职能。人民政协不是国家机关。
17. 居委会和村委会成员都是 <span style="color:red; background:yellow">由居民直接投票选举</span> 产生的，有利于人民群众<span style="color:red; background:yellow">直接</span>行使民主权利。
18. 公有制经济包括 <span style="color:red; background:yellow">国有经济、集体经济</span> 以及混合所有制经济中的<span style="color:red; background:yellow">国有</span>成分和 <span style="color:red; background:yellow">集体</span> 成分。国有经济是国民经济的 <span style="color:red; background:yellow">主导力量</span>。在我国，公有制经济和 <span style="color:red; background:yellow">非公有制</span> 经济是 <span style="color:red; background:yellow">社会主义市场经济</span> 的重要组成部分。<span style="color:red; background:yellow">市场</span> 在资源配置中起决定性作用的经济体制(看不见的手)。我国社会主义市场经济体制把 <span style="color:red; background:yellow">社会主义制度</span> 和<span style="color:red; background:yellow">市场经济</span>有机结合起来，充分发挥<span style="color:red; background:yellow">市场</span>在资源配置中的决定性作用，更好发挥 <span style="color:red; background:yellow">政府</span> 作用，进行科学宏观调控。
19. 国家 <span style="color:red; background:yellow">行政</span> 机关是国家 <span style="color:red; background:yellow">权力</span> 机关的执行机关。我国最高行政机关是 <span style="color:red; background:yellow">国务院（中央人民政府）</span>。
20. 监察委员会是行使国家<span style="color:red; background:yellow">监察职能</span> 的专责机关，对所有行使<span style="color:red; background:yellow">公权力</span>的公职人员进行监察。<span style="color:red; background:yellow">调查公职人员涉嫌职务违法和职务犯罪</span> 是监察委员会的一项经常性工作。<span style="color:red; background:yellow">监督</span> 是监察委员会的 <span style="color:red; background:yellow">首要</span> 职责。
21. 我国的司法机关包括人民 <span style="color:red; background:yellow">法院</span> 和人民 <span style="color:red; background:yellow">检察院</span>。人民<span style="color:red; background:yellow">法院</span>是国家的 <span style="color:red; background:yellow">审判</span> 机关，独立行使<span style="color:red; background:yellow">审判</span>权。人民<span style="color:red; background:yellow">检察院</span>是国家的 <span style="color:red; background:yellow">法律监督</span> 机关，独立行使<span style="color:red; background:yellow">检察</span>权。
22. <span style="color:red; background:yellow">公正司法</span>是捍卫社会公平正义的最后一道防线。
23. <span style="color:red; background:yellow">公平正义</span> 是法治社会的核心价值。
24. 司法机关必须坚持以 <span style="color:red; background:yellow">事实</span> 为根据，以 <span style="color:red; background:yellow">法律</span> 为准绳。
